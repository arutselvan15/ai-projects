# Generative AI project lifecycle

## Project Lifecycle

### Scope

Define the use case

### Select

Choose an existing model or pretrain your model.  In general you start with existing model.

### Adapt and align model

- Prompt Engineering
  - In-context learning
    - one or few shot learning
- Fine-tuning
- Align with human feedback (reinforcement learning)
- Evaluate (used in every step)
  - metrics
  - benchmark

### Application Integration

- Optimize and deploy model for inference.
- Augment model and build LLM powered applications

## Team Roles


### Software Engineer
    - Resonsible for writing software application
    - Someone has learned basics fo LLM/Prompting

### Machine Learning Engineer
    - Responsible for implementing AI systems
    - Ideally familiar of LLMs/Prompting, RAG, Fine-tuning

### Data Scientist
    - Responsible for analyzing data to make recommendataions to guide project or business decisions

### Data Engineer
    - Responsible for organizing data and ensuring data quality

### Product Manager
    - Responsible for identifying and scoping the project
