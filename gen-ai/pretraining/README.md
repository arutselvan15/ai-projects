# Pretraining

Pretraining general-purpose LLMs by learning from internet text.

- takes $10s of millions
- many months
- huge amount of data

Requires highly specialized domain, only big techs can do.

## When to do?

When you have specific need and have domain and cost to build.

Bloomberg's 50-billion parameter LLM model, purpose built from scratch for finance.


## How to choose a model?

    Parameters          What it does                                            Use case
    1B              pattern matching & basic knowledge                          restaurant review sentiment
    10B             greater world knowledge, can follow basic instruction       food order chatbot
    100B+           rich world knowledge, complex reasoning                     brainstorming partner
